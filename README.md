🌦️ Weather Forecast Web Application

📌 Project Overview

The Weather Forecast Web Application is a web-based project developed using Django and Bootstrap that provides real-time weather information for any city. The application fetches live weather data using the OpenWeather API and displays details such as temperature, humidity, weather conditions, and location in a clean and user-friendly interface.

🚀 Features
	•	Search weather by city name
	•	Displays temperature, humidity, wind speed, and weather condition
	•	Real-time data using OpenWeather API
	•	Responsive and attractive UI using Bootstrap
	•	Error handling for invalid city names
	•	Fast and lightweight web application

🛠️ Technologies Used
	•	Python
	•	Django Framework
	•	HTML, CSS, Bootstrap
	•	OpenWeather API
	•	JavaScript (for UI enhancements)

⚙️ How It Works
	1.	User enters a city name in the search box
	2.	Django backend sends a request to OpenWeather API
	3.	API returns real-time weather data
	4.	Data is processed and displayed on the web page

📂 Use Cases
	•	Learning Django and API integration
	•	Academic and PBL projects
	•	Real-time weather information access
	•	Beginner-friendly full-stack project
